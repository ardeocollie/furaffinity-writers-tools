# FurAffinity Writer's Tools
A collection of Python scripts to help writers publish their stories to FurAffinity

## Tools

### TXT Formatter
Converts the files out of your favorite word processor into a nicely formatted .txt for FurAffinity's in-site reader.

Supported formats:
- .docx
- .odt

### Thumbnail Builder
Creates thumbnails from a background image and a set of keywords.
